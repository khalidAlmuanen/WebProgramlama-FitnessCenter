# FitnessCenterApp

Bu proje, Sakarya Üniversitesi WEB PROGRAMLAMA dersi kapsamında geliştirilen
bir spor salonu randevu ve vücut dönüşüm takip uygulamasıdır.

- ASP.NET Core MVC
- Entity Framework Core
- Identity ile kimlik doğrulama
- Randevu yönetimi, eğitmen uygunluğu
- AI destekli egzersiz önerileri ve vücut dönüşüm tahmini
